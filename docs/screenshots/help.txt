Unified code quality checker for Rust. Headless-first, JSON output, CI-ready.

Usage: codemetrics <COMMAND>

Commands:
  check            Run all quality checks and report results
  crap             CRAP metric only
  debt             Technical debt only
  doccov           Documentation coverage only
  dupfind          Code duplication only
  complexity       Cyclomatic complexity report
  init             Generate default config file
  run              Run all quality tools in batch mode using .quality.toml config
  history          Record or display quality metrics history
  install-hooks    Install a quality pre-commit git hook
  uninstall-hooks  Remove the quality pre-commit git hook
  watch            Watch for file changes and re-run relevant quality checks
  discover         Discover available quality tools and their capabilities
  help             Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
